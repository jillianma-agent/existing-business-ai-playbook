# Font Pairings

Curated font pairings for minisites. Default to the monday.com system fonts unless the user specifies otherwise.

## monday.com Default

| Role | Font | Weight | Use |
|---|---|---|---|
| Headings | Figtree | 600-700 | Page titles, section headings |
| Body | Figtree | 400-500 | Paragraphs, descriptions, UI text |
| Accent | Figtree | 300 | Large hero numbers, light captions |

```html
<link href="https://fonts.googleapis.com/css2?family=Figtree:wght@300;400;500;600;700&display=swap" rel="stylesheet">
```

This is the default for platform-related content and digital pages.

## monday.com Marketing

| Role | Font | Weight | Use |
|---|---|---|---|
| Display | Poppins | 600-700 | Marketing headlines, brand statements |
| Body | Figtree | 400-500 | Supporting text |

```html
<link href="https://fonts.googleapis.com/css2?family=Poppins:wght@300;400;600;700&family=Figtree:wght@400;500;600&display=swap" rel="stylesheet">
```

Use for marketing landing pages and brand-forward content.

## Alternative Pairings

For non-monday branded pages when the user wants a distinctive look:

### Technical / Developer-focused

| Role | Font | Character |
|---|---|---|
| Display | Space Grotesk | Geometric, modern, technical feel |
| Body | Inter | Clean readability, wide language support |

```html
<link href="https://fonts.googleapis.com/css2?family=Space+Grotesk:wght@400;500;600;700&family=Inter:wght@400;500;600&display=swap" rel="stylesheet">
```

### Editorial / Content-heavy

| Role | Font | Character |
|---|---|---|
| Display | Outfit | Warm geometric, friendly authority |
| Body | Source Sans 3 | Excellent readability at small sizes |

```html
<link href="https://fonts.googleapis.com/css2?family=Outfit:wght@400;500;600;700&family=Source+Sans+3:wght@400;500;600&display=swap" rel="stylesheet">
```

### Bold / Impact-driven

| Role | Font | Character |
|---|---|---|
| Display | Plus Jakarta Sans | Strong, confident, modern |
| Body | DM Sans | Clean, balanced, professional |

```html
<link href="https://fonts.googleapis.com/css2?family=Plus+Jakarta+Sans:wght@400;500;600;700;800&family=DM+Sans:wght@400;500;600&display=swap" rel="stylesheet">
```

### Warm / Approachable

| Role | Font | Character |
|---|---|---|
| Display | Nunito | Rounded, soft, inviting |
| Body | Lato | Warm but professional |

```html
<link href="https://fonts.googleapis.com/css2?family=Nunito:wght@400;600;700;800&family=Lato:wght@400;700&display=swap" rel="stylesheet">
```

## Selection Guide

| Page type | Recommended pairing |
|---|---|
| monday.com platform content | monday Default (Figtree) |
| Marketing landing page | monday Marketing (Poppins + Figtree) |
| Developer documentation | Technical (Space Grotesk + Inter) |
| Blog / long-form content | Editorial (Outfit + Source Sans 3) |
| Product launch / announcement | Bold (Plus Jakarta Sans + DM Sans) |
| Community / educational | Warm (Nunito + Lato) |
